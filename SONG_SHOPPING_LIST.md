## How to add a new recipe once a file lands

Edit `stemforge-demo-material/recipes.yaml` and append an entry. Template:

```yaml
  - recipe_id: w05-xtal-microtune-demo
    lesson: w05-aphex-tuning
    cmd: stemforge split {input} --pipeline default --no-slice
    input: ${stemforge_root}/grooves/01 - Aphex Twin - Xtal.wav
    renders:
      drums: build/audio/stemforge-renders/w05/xtal_drums.wav
      bass:  build/audio/stemforge-renders/w05/xtal_bass.wav
    teaching: Real Aphex vs our Express Yourself microtuning demo — A/B the kick-as-bass signature.
```

Then `python3 tools/stemforge_runner.py --recipe w05-xtal-microtune-demo`.
