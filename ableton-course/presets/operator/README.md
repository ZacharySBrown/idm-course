# Operator Presets

These are Ableton .adv preset files used by the render pipeline.
They must be committed to the repo so rendering is reproducible.

To export a preset from Ableton:
1. Build the patch in Operator
2. Right-click the device title bar → Save Preset
3. Navigate to this directory and save

Naming convention: <episode-id>_<description>.adv
Example: e01_aphex_metallic_hit.adv
