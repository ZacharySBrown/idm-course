# Mutation 1 — drum swap

**Rule:** replace the drum pattern wholesale. Keep everything else identical.

Two clean paths in Live 12:

**Path A — MIDI Recombine.** Select the drum MIDI clip. Right-click → Transform → **Recombine → Shuffle**. Live rotates and reshuffles the hits. Same kit, same velocities, new order. <Citation bib="ableton_live_12_manual" />

**Path B — Drum Rack hot-swap.** Keep the MIDI, swap the entire Drum Rack. Same notes, unfamiliar voices. Spec A §5.4 permits either; Path B generates more contrast per minute.

**Limitation.** Drum-swap variations can feel like the drummer quit mid-song if you do not bridge them. Insert a one-bar crash or reversed cymbal on the bar before the swap. The edit point is where this mutation lives or dies.

Do not tune the new kit to the track. That was Week 5. This week you are moving fast.
