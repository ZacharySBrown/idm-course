# Ship it — five mutations, six minutes, one bounce

Six minutes. One bounce. No second pass.

**Steps:**

1. Pull your best 16-bar loop from any week so far. Not a new one.
2. Arrangement View. Drop locators: **0:00 Intro, 0:45 A, 2:00 B, 3:15 Reduction, 3:45 Peak, 5:00 Outro**. <Citation bib="ableton_live_12_manual" />
3. Duplicate the loop five times, one per named section between bookends.
4. Apply the five mutations, one per duplicate, in an order that creates an arc:
   - **Drum swap** (Recombine → Shuffle, or swap the Drum Rack)
   - **Bass seed** (Seed ±7, Capture MIDI)
   - **Filter sweep** (Auto Filter + macro, unlinked envelope)
   - **Reverse** (one element only)
   - **New layer** (one sound absent from the original)
5. Save `~/IDM_COURSE_versions/2026-04-20_w10_track_01.als` **before** bouncing.
6. Bounce stereo, 44.1/24-bit, `w10_<yourname>_five-mutations.wav`.

**Write this down:** one sentence per mutation — which landed, which felt forced. <Citation bib="rick_rubin_creative_act" />

No wrap-up. Ship.
