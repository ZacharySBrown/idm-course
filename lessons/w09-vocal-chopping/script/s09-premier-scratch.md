[pause 200ms] DJ Premier, and specifically *Ten Crack Commandments*, Notorious B.I.G., 1997. The hook is a scratched voice sample — Chuck D counting numbers — and the scratched performance *is* the sample. Premier, on the record in interviews: ~"That loop is me sampling my scratching by hand. It was me experimenting, and it sounded dope."~

[pause 300ms]

This is a different method than everything else this week. Prefuse chops then finger-drums. Dilla chops then rearranges. Kanye chops then pitches. Premier performs *first*, then samples the performance, then triggers it as if it were a drum.

Stripped to verbs it goes:

One. Perform a pass on a sound. The pass itself is the sample source. Turntable, a MIDI keyboard retriggering a vocal, a tape-stop plugin, a reverse gate abused — whatever has a performance hand on it.

Two. Bounce the pass.

Three. Load the bounce into a sampler and trigger it as percussion. Not as a melody. Not as a loop. As a drum. Premier triggered that scratched Chuck D loop on a single pad, rhythmically, the way you would trigger a snare.

[pause 300ms]

For the Hot Pants vocal, the Ableton translation is pleasingly simple. Take the acapella bounce. Put it on an audio track. Record yourself dragging the clip start — the warp handle — in real time while the track plays. You are writing a clip envelope live. Record a minute of this. The result is a performance where the vocal jumps around, stutters, reverses briefly, stalls. Imperfect. Bounce it.

Now load that bounce into Simpler in Classic mode, not Slicing. Map it to a single pad. Trigger that pad from a drum pattern, as if the entire bounce were a one-shot. The vocal performance now behaves like a rhythmic instrument.

[pause 200ms] Premier's actual gear was an MPC60 and two Technics 1200s. The method is not gear-dependent — any sampler will do. The trick is the hand performance first, bounce second, trigger as drum third. [pause 300ms] Anybody can program a drum pattern. Very few producers think to *perform* the sample before they sample it.
