# Technique continued — the 2.5-second constraint as composition force

The RAM division forced specific compositional choices that defined the era:

- **Chops not loops.** You cannot load an 8-bar break at 26.04 kHz in 2.5 seconds; you load the first kick + snare + one hat and trigger each as separate pads. Wu-Tang's "Method Man" and Nas's "N.Y. State of Mind" drum programming is direct evidence.
- **Short samples, retriggered.** The "stabby" feel of SP-1200-era hip-hop is a RAM-limit artifact: every hit is a short sample, not a sustain.
- **45-rpm sampling to steal headroom.** Playing records at 45 rpm halves sample duration. RZA and Pete Rock both did this constantly (`bib:charnas_dilla_time`).
- **Global swing only.** Unlike the MPC3000's per-pad swing, the SP-1200 had one swing value for the whole pattern. This is why <term key="dilla_time">Dilla time</term> is not an SP-1200 aesthetic — the architecture doesn't support it.

The result: SP-1200 records sound *tight*, *stabby*, *aliased*, *bandwidth-limited*. MPC3000 records sound *loose*, *wide*, *punchy*. Same producers could make both; the machine chose the feel.
