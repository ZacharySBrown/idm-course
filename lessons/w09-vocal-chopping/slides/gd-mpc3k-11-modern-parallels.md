# Modern workflows — who's still using this approach

You do not need an MPC3000 to make <term key="dilla_time">Dilla time</term>. You need the *discipline*. Producers who ported the method:

- **Knxwledge, Kaytranada, Mndsgn** — Drum Rack + per-chain groove + deliberate off-grid nudging. All on record as Dilla disciples.
- **Madlib, post-MPC** — moved to Boss SP-303 and later laptop workflows, kept the hand-programmed, quantize-off ethic (`bib:charnas_dilla_time`).
- **The entire LA beat scene (Flying Lotus, Teebs, Samiyam)** — MPC60/2500/3000 + Ableton hybrid. The MPC sequences; Ableton arranges. The feel stays on the hardware.
- **Ableton Push 3 standalone** — pads + step sequencer + per-pad swing via Groove Pool makes it the closest modern all-in-one (`bib:ableton_push_3_docs`). It is not an MPC. It is close enough for the feel.

The technique survives the hardware. The feel is in the nudges, not the box.
