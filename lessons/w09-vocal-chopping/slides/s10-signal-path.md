# The chopped-vocal signal path, committed

The common path for all four methods this week, expressed in Ableton verbs (`bib:ableton_live_12_manual`):

1. **Stemforge bounce** — load `build/audio/stemforge-renders/w09/hp_vocals.wav`. Freeze the track. Flatten. The LALAL output is the source-of-truth (`bib:stemforge_repo`, recipe `w09-hot-pants-vocal-stem`).
2. **Slicer front-end** — Simpler in Slicing mode, 1/16 for chipmunk, 1/32 for Dilla micro-chop, By Transient for Prefuse. Warp mode Re-Pitch when transposing. Classic when you want SP-1200-style pitch-time coupling.
3. **Per-chain character** — Little AlterBoy on individual chains for the "eh-ah-oh" doubles, Formant ±5 Pitch ±2 (`bib:soundtoys_little_alterboy_manual`).
4. **Group width** — MicroShift Style II, Detune 100%, Delay 100% across the chopped-vocal group (`bib:soundtoys_microshift_manual`).
5. **Commit rule** — every time you print a pass, delete the unused source track. The ladder only works if you can't go back.

One track in. One bounce out. One commit per pass.
