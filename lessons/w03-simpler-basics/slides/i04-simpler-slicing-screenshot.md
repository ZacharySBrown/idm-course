# Simpler in Slicing mode — waveform with numbered markers

The screenshot shows Simpler with Slicing selected. The waveform is overlaid with numbered vertical markers, one per slice. The slice list sits to the right of the waveform with per-slice gain.

Controls that matter:

- **Slice by** dropdown — Transient / Beat / Region / Manual. The choice writes the markers.
- **Sensitivity** (Transient only) — higher detects more onsets, fewer markers get merged.
- **Playback** — Mono / Poly / Thru. The polyphony behavior between slices.
- **Slices** list — right-hand column — each slice's start time, gain, and delete button. Reorder by dragging.
- **Cmd+I** (Manual mode) — insert a new marker at the current cursor position.

Slicing mode is where you spend time when you want fast chopping without spawning a full Drum Rack. For per-slice processing, use Slice to New MIDI Track instead.
