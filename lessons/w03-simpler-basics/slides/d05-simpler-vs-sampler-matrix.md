# Simpler vs Sampler — the feature matrix

A decision matrix, not a ladder. Ableton's marketing positions Sampler as the grown-up Simpler. The matrix below disagrees.

| Feature | Simpler | Sampler |
|---|---|---|
| Warp engine | yes (Beats / Tones / Texture / Complex / Complex Pro / Re-Pitch) | no |
| Three modes (Classic / One-Shot / Slicing) | yes | no — one mode, multi-sample |
| Multi-sample zones | no | yes (up to 1024) |
| Round-robin / random-robin | no | yes |
| Velocity-split keymaps | no | yes |
| Key-tracked mod depth | limited | yes, across mod matrix |
| Modulation matrix | 2 LFOs + envelopes | full matrix, multiple LFOs, key/velocity/random sources |
| Pad-friendly drum workflow | yes (One-Shot is built for it) | clumsy |

Rule of thumb for this course: Drum Rack pads hold Simpler unless you need round-robin. Vocal chops live in Simpler Classic. Sampler shows up when we build multi-layer snares in Week 6.

They are different tools. The suite level is a licensing accident.
