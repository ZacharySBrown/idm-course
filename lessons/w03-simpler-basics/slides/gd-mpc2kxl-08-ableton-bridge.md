# Ableton bridge — Drum Rack + Simpler Slicing + Resample track

The Ableton equivalent of the MPC ladder. Parameter table below.

**Screenshot** (to be supplied in Wave 5): a Live set with three tracks — a source audio track with the vocal phrase; a MIDI track named "Chop 1" with a Drum Rack full of Simpler-per-pad chains from Slice to New MIDI Track; a resample audio track set to record from "Chop 1".

| MPC2000XL | Ableton Live 12 equivalent | Notes |
|---|---|---|
| 16 velocity-sensitive pads | Drum Rack, 16 visible pads (one octave) | Page Up/Down scrolls the other 96 |
| Bank A/B/C/D | Page Up/Page Down on Drum Rack | Each page is 16 pads, 8 pages = 128 pads |
| Chop Shop (Regions) | Slice to New MIDI Track → Slice to Beat Division or Transient | Region count maps to 1/16 count for a bar |
| 16-Levels | Simpler Classic mode + chromatic MIDI clip | Load sample, play across keyboard |
| Note Variation slider | Drum Sampler's Velocity Shaper + Time Warp | Per-pad in Live 12 |
| Note Repeat | Push 3 Note Repeat, or Ableton Note Echo MIDI effect | Push is the closest feel-match |
| Resample | Audio track, Input = "Resample", record-arm | No button, but the plumbing is the same |
| TC 1/16 swing 54–58% | Groove Pool: MPC-58 swing template | Apply per-clip, not globally |

The bridge is faithful but not identical. Swing curves especially — Ableton's grooves are linear-time, not the MPC's chip-native curve.
