# Creative uses beyond chopping vocals

The MPC2000XL was designed for hip-hop but its architecture supports more than the Prefuse workflow. Four uses worth knowing.

- **Sample-and-hold synthesis**. Record 500 ms of a field recording into a pad. Set the pad to loop across its full length. Play chromatically across 16-Levels. You now have a wavetable-adjacent pitched instrument built from one captured sound. Aphex Twin's *Druqks*-era textures used this on a different Akai (S950), but the move transfers.
- **Beat-sliced drum programming with no MIDI input**. Sample a 1-bar break. Chop by threshold. Drop the slices onto pads. Sequence a new 16-step pattern by entering notes in Step Edit mode — no tapping. The tracker-adjacent workflow that makes Autechre-style patterns possible on a non-computer.
- **Resampling as freeze**. The MPC has no track freeze. Resampling the output of a CPU-heavy track — layered samples with filter sweeps — gives you a single flat sample you can treat as source. This is how you stack 60 pads of processing into a machine with 16 voices.
- **Live finger-drumming over DJ sets**. Bank A drums, Bank B one-shots, Bank C vocals, Bank D FX. A MIDI clock in from a turntable rig keeps sequencer-based patterns locked to the set tempo. This is what Herren does on stage.
