# Three modes, one device

Simpler has three modes. The tabs sit above the waveform. Same device, three jobs.

- **Classic** — melodic play. Pitch tracks the keyboard; Warp keeps time locked at any transposition. Amp envelope, filter, LFO. This is the mode for chromatic chopping — one vocal chunk across sixteen keys.
- **One-Shot** — a single sample fires start-to-end on each note. No loop, no release tail. Meant for drum hits. Fade In / Fade Out sliders, Trigger vs Gate, Snap-to-zero-crossing. This is where the kicks live.
- **Slicing** — Simpler chops the sample into numbered segments. Each segment maps to its own MIDI note. Transient / Beat / Region / Manual decide where the markers fall.

You pick the mode to match the job. Loading a drum hit into Classic mode and triggering from C3 is a mistake you make exactly once.
