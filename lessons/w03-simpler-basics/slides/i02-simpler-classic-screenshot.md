# Simpler in Classic mode — the full instrument

The screenshot shows Simpler with Classic selected. Waveform on top, three mode tabs above it, <term key="amp_envelope">amp envelope</term> and filter and LFO sections below.

This is the mode most new users stay in for every job, which is a mistake. The filter and envelope are useful for melodic sources — pitched vocal chunks, a flute stab, a synth sample — but they add complexity when you just want a drum hit to fire.

Key controls visible in the screenshot:

- **Voices** — Mono / Poly / Legato at the top right. Set to Mono for bass sources, Poly for pads, Legato for portamento leads.
- **Warp** toggle — leftmost of the three tabs above the waveform. On by default in Classic.
- **Filter** section — 4-pole LP24 Moog-style by default. Multimode with key-tracking.
- **LFO** — one, routable to pitch / filter / amp / pan.
