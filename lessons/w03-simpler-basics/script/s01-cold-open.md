Before we open Ableton — a small admission.

[pause 400ms]

Nine out of ten Drum Racks you will build over the next year are sixteen instances of *Simpler*. Not Sampler. Not Operator. Not a boutique sampler you bought in a sale and forgot. Simpler — the device Ableton ships in every tier, including Intro, with three mode buttons stacked above a waveform that barely fits the device view.

[pause 300ms]

This is a problem for the tutorial economy, because Simpler is not glamorous. There is no oscilloscope. There are no unison voices with drift. There is a filter, an amp envelope, an LFO, and a warp switch. ~That is the entire party.~

But here is the receipt. Tape Notes episode 140 — Kieran Hebden, the Four Tet episode, 2024. Hebden says, quote, *"I'm all about the mouse."* He edits audio directly on the timeline. He does not use Simpler. He does not use Sampler. He does not use any hardware sampler. He cuts bounces in Arrangement view and calls it a day.

[pause 500ms]

Fine. Hebden is working in a lane where every sound in a track is already a committed, bounced, pre-edited audio chunk. That is one valid way to work. It is not the way you work when you are *triggering* hits from a MIDI pattern — when you want to program sixteenths on a kick and have each hit be identical, or have each hit be slightly un-identical on purpose.

For that — and for almost everything this course does with drums — Simpler is the instrument.

[pause 300ms]

So this week we stop skipping past it. We open Simpler and read the three mode tabs. We figure out which mode handles a drum hit, which mode handles a chromatic vocal chop, and which mode slices a break. We walk through the *Slice to New MIDI Track* command, which is the move most tutorials either botch or skip.

Then we open Drum Rack and learn what it actually is — which is not a sampler, it is a grid of independent device chains with MIDI notes routed to each one.

Then we use the one thing stemforge built for exactly this week — the *w03-express-oneshots* recipe, which pulls per-drum-type one-shots out of Charles Wright's *Express Yourself* using LarsNet classification. We drop those into a Drum Rack, program a four-bar pattern, and bounce.

The Ableton Live 12 Reference Manual is open beside me the whole time. Cite that, not me.
