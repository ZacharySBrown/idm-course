# The five controls that matter on the Monologue panel

Ignore the LFO, the amp envelope, the drive — every mid-range analog has those. The microtuning-specific controls, in order of how you use them: <Citation bib="korg_monologue_afx_manual" />

- **EDIT MODE button** — hold + turn the program dial → parameter 17 is **Microtuning**. Here you pick from 12-TET, equal-temperament user 1–6, or "scale user" 1–6. Six slots of user microtuning per program.
- **Microtuning edit page** — accessible via EDIT MODE + key group. Per-key cent offsets from −75 to +75, in 1-cent steps. This is the heart of the machine: a 25-key piano where every key carries its own cent offset.
- **PROGRAM EDIT → SLIDE TIME** — separate per-patch, affects how microtuned targets glide. Short slide + microtuning = the "chopped, slightly wrong" Aphex lead.
- **Sequencer STEP mode** — per-step microtuning *and* parameter motion. A single step can play the current scale's 7-limit third; the next step can detune that by an additional motion-sequenced 15 cents.
- **MICROTUNING COPY** — SysEx-dumpable. This is how `.scl`-equivalent data moves between the Monologue and a DAW; you need Korg's Sound Librarian + the MTS-ESP conversion chain.

Everything else is wallpaper.
