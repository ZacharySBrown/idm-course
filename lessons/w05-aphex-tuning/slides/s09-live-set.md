# Live set — w05-drum-tuning-demo.als

The demo session lives at `live-sets/w05-drum-tuning-demo.als`. Open it before the exercise.

What's in it:

- One muted audio track — the dry `express_dry.wav` reference, for A/B.
- One MIDI track with a Drum Rack — kick on C1, snare on D1, hat on F#1. Simpler Classic on each pad.
- Locators at 0:00 (dry) and 0:15 (tuned). Matches the exercise geometry.
- A reference marker showing the track's root (E1).

The tuning values are deliberately wrong at session start. Your job is to move them. Transpose the kick to land at ~E1. Detune the snare to the fifth.

Bounce via **Export Audio/Video** or Freeze-and-Flatten the Drum Rack. <Citation bib="ableton_live_12_manual" />

The session is a scaffold, not a solution. The solution is your ear.
