# Exercise B — build your own microtuning, then write against it

Past replication. Compose your own scale — 17-EDO or custom JI — and write a 30-second bass riff that works in it.

**Setup (15 min):**
1. In Tuning Systems, create a new user scale. Pick one:
   - **17-EDO** — twelve of the seventeen steps, distributed as a "5-limit pseudo-diatonic": steps 0, 3, 5, 7, 9, 12, 14 (cents: 0, 212, 353, 494, 635, 847, 988).
   - **Custom 7-limit JI** — your own seven pitches using only ratios with prime factors ≤7. Example: 1/1, 9/8, 7/6, 4/3, 3/2, 7/4, 2/1.
2. Save. Load into Tuning Systems.

**The move (30 min):**
3. New Operator track. Simple sine patch. Write a 30-second bass riff — eight bars, root on the 1, scale-degree landmarks on the 3, 5, 7 of each bar.
4. Add a kick tuned to root (same Simpler-into-Tuning-Systems move from Exercise A).
5. Bounce.

**Deliverable.** The 30-second WAV + one paragraph: *what intervals in your scale felt inevitable, and which felt like you were fighting them?* The inevitable ones are doing the compositional work. <Citation bib="aphex_pitchfork_microtuning" />

Forty-five minutes total. Commit. Next week we stop tuning and start feedback.
