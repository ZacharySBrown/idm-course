# Korg Monologue (AFX edition) — the microtuner with James' name on it

Released January 2017. Monophonic analog, two VCOs, one filter, sixteen-step sequencer, built-in **microtuning** per-patch. The AFX-branded firmware ships with six user scales programmed by Richard D. James himself — loadable per-preset. <Citation bib="korg_monologue_afx_manual" />

The Monologue is not the interesting box. Every mid-range analog synth in 2017 had a monophonic engine and a 16-step sequencer. The interesting box is the one Korg engineer **Kazuki Saita** built to hand-deliver custom MIDI microtuning to Aphex's rig — and which got folded back into the Monologue's public firmware as the hook. <Citation bib="aphex_syrobonkers_interview" />

The consumer product is a memorial to a one-off. You can buy the memorial for $300. The original is still in James' studio.

This deep-dive: how the microtuning actually works, what "RDJ-designed scales" means, how to translate it to Ableton without buying the hardware.
