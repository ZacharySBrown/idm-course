# The rub — two harmonic fields in the same bar

- <term key="the_rub">The rub</term> is not dissonance. Dissonance is a semitone collision inside one harmonic field. The rub is two *different* fields pulling on the same bar.
- Canonical example (`bib:charnas_dilla_time`, Ch. 14): Dilla's *Get Dis Money* samples a Herbie Hancock vocal that implies one mode; Dilla writes a bassline in the Aeolian. Both are musical. Neither resolves to the other.
- **What the diagram shows:** top row — sample's implied mode (Mixolydian, say, because the sampled horn has a b7). Bottom row — Dilla's bassline mode (Aeolian, with a b6 the sample does not contain). The shared pitch classes are the pivot; the non-shared ones are the friction.
- The ear tries to settle on one of the two. It cannot. That unresolved tug is the hook.
- **Rule:** ≥2 non-shared pitch classes between sample and bass to get a rub. 0 is parallel. 1 is accidental. 3+ usually reads as a mistake.
