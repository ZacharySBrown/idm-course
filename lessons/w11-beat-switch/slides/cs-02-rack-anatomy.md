# Rack anatomy — what you are looking at

- Open any Instrument Rack or Drum Rack. Click the "Show Chain List" button (vertical chain icon, left edge of the Rack). The chain list appears. Above it: the **Chain Selector ruler** — a thin strip with a value from 0–127.
- **Four controls that matter for a switch:**
  1. **Chain Selector ruler** — drag the yellow triangle to set the current selector value. Whatever zones that value falls inside are active.
  2. **Chain zone bar** (one per chain) — a range with two edges. Default is 0–127 (always on). Narrow it to, e.g., 0–63 for "active when selector ≤ 63."
  3. **Macros bar** (top of Rack) — 8 or 16 macros. Assign Chain Selector to one macro via right-click → Map.
  4. **Macro Variations button** (new in 12.2, right side of macros bar) — opens the snapshot drawer. Store, name, recall.
- `bib:ableton_live_12_manual`, *Instrument Rack → Zones*. Zones have fade regions (the darker outer edges) that crossfade between adjacent chains; the hard-switch move uses zero-width fades.
