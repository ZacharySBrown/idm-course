The confession. Every author of this kind of course — including the one writing this one — will at some point tell you that gear does not matter, and then, ten minutes later, publish a paragraph listing eight specific pieces of gear by model number and year. This course will do that constantly. It will name the Yamaha QY700. The Akai S950. The Dorrough 610. The Eventide Orville. The Korg Monologue. It will name them because the specificity is *useful* — you can buy a QY700, you cannot buy "vintage vibes" — but the specificity also creates a lie. The lie is that if you owned the gear you would sound like the record.

The honest version is: gear *mostly* does not matter, except when it does, and the skill is noticing which case you are in. Tape Op issue eighty-nine, the Squarepusher interview — Jenkinson is asked about the QY700, and he mostly answers about *why* he refuses a computer, not why he chose the sequencer. The answer is: because he does not want a screen. The QY700 is incidental to the argument. Any sequencer without a big screen would have worked.

Now the aside, Sound on Sound style. Because Sound on Sound is where half of this course's citations live, and they have a specific voice — technical, matter-of-fact, vaguely resigned, willing to explain how a thing actually works without selling you on it. The aside:

Ableton added a *Reference track* feature to Live Twelve's mixer. It is a button. You drop a file on it and it plays, level-matched, against whatever you are working on. The reason this feature exists is that the market for third-party reference plugins — Reference 4 is the canonical one — made it obvious that engineers had been doing this manually in every DAW for years. Ableton did not invent the protocol. Ableton added a convenience for a protocol that predates the button by approximately thirty years. The button is fine. Do not mistake the button for the skill.

[pause 400ms]

This week, you do not need the button. An unmuted audio clip on an adjacent track is the same protocol with fewer abstractions and zero license costs.
