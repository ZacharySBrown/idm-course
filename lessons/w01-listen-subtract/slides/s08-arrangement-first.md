# Pre-place the locators, then close the laptop

A session template, not aspirational, load-for-every-session:

- Locators at **0:00 intro / 0:45 A / 2:00 B / 3:00 breakdown / 3:45 peak / 5:00 outro / 5:45 end.**
- A reference track on a muted audio lane. One Aphex, one Autechre, one Dilla, one Kendrick. A/B every 20 minutes.
- Drum / Bass / Melody / Texture / Vocal groups pre-grouped in the Session.
- One return for plate, one for tape delay, one for granular, one for Roar feedback. You are not using them this week — they are there so you do not spend the next 45 minutes building them.

The locators are suggestions, not contracts. The point is that placing them converts the session from open-ended into *finite*. <Citation bib="ableton_live_12_manual" />

Then close the laptop lid, route to speakers, and write notes on paper. <Citation bib="sos_autechre_april_2004" />
