# Ship the listening

The deliverable. Thirty seconds. No synths. No processing. This is a listening exercise and it stays a listening exercise.

**Steps:**

1. Load `benjamins_drums.wav`, `benjamins_bass.wav`, `benjamins_vocals.wav`, `benjamins_other.wav` on four adjacent audio tracks. Align to bar 1. <Citation bib="stemforge_repo" />
2. Solo each for 20 seconds. Write *one sentence* per stem on paper — what it is doing rhythmically, what it is doing harmonically.
3. In Arrangement View, place a locator at 0:00 and 0:15. From 0:00 to 0:15, mute the drums track. At 0:15 un-mute it. Everything else plays through.
4. Bounce the result as a 30-second stereo file named `w01_<yourname>_mute-ab.wav`. 44.1 kHz / 24-bit. <Citation bib="ableton_live_12_manual" />

**Deliverable:** the WAV and the four one-sentence notes. Post both. That is Week 1.

No triumphant wrap-up. Go listen.
