# The densest bar, as master state

Subtraction inverts the usual additive workflow. You do not build the intro first. You build the densest bar first — chorus, drop, peak — and treat every other section as a *mute* of that state.

- Intro — everything muted except one texture.
- A — drums unmute.
- B — bass joins.
- Breakdown — drums drop, texture returns.
- Peak — master state, all unmuted.
- Outro — inverse of the intro.

This is finish-friendly because the arrangement becomes a mute-automation problem, not a sound-design problem. Rubin, in *The Creative Act*, frames it as removing everything that is not the song. <Citation bib="rick_rubin_creative_act" />
