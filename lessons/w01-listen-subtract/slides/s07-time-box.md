# The 45-minute rabbit hole

A rule that sounds arbitrary and is not: **every new sound gets 45 minutes, then you commit or you move on.** Kitchen timer on the desk. When it rings, bounce-in-place or freeze.

This is not a productivity hack. It is a workflow that assumes you have already lost the other version of this fight. Separate the four mentalities into four separate sessions:

- **R&D** — sound design. Save presets. Nothing used yet.
- **Writing** — only patches already saved. No new synths.
- **Arrangement** — monitors down, Arrangement View full-screen, laptop lid closed.
- **Mixing** — separate pass after arrangement is locked.

Mixing *inside* a loop is the canonical intermediate-producer mistake. Every curriculum says so. <Citation bib="oblique_strategies_eno" /> — Eno keeps the cards on the desk for this moment.
