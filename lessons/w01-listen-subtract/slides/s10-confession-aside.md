# A confession, and one aside

Confession first. Every author of this kind of course will tell you gear does not matter and then publish a paragraph naming eight pieces of gear. The honest version: gear *mostly* does not matter, except when it does, and the skill is noticing which case you are in. <Citation bib="tape_op_89" /> — Jenkinson, asked about the QY700, answers mostly about *why* he refuses a computer rather than why he chose the sequencer.

Aside, Sound on Sound-style. Ableton's Reference track control in Live 12's mixer was shipped because Reference 4-style plugins had made it obvious engineers were already doing this manually. The protocol predates the button by thirty years. The button is a convenience. <Citation bib="ableton_live_12_manual" />

You will not need the button this week. An unmuted audio clip is the same protocol with fewer abstractions.
