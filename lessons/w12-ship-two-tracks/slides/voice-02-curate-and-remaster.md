# Curate four, re-master as a set — the cohesion pass

You have circled four tracks from the curation exercise. Now treat them as an EP, not as four singles.

The cohesion pass is about *relative* loudness, tone, and space — not absolute numbers per track. Four tracks that each hit −14 LUFS on their own will still sound mismatched if one is compressed to death and one has 14 LU of dynamic range. Fix that before it ships.

Protocol for the four-track pass.

1. Bounce the un-mastered mix of each of the four tracks. Drop all four on adjacent Live tracks.
2. Level-match all four to the same integrated LUFS with Utility (mast-05 protocol).
3. Play them back-to-back, 30 seconds each, in the proposed EP order. Write down what is *missing* on each track relative to the others — not what is wrong in isolation.
4. Common gaps: one track has less sub-energy, one has more top-end, one has a noticeably different stereo width. These are the translation targets for the re-master.
5. Re-master each track individually with the same master chain (mast-04) but with the cohesion gaps in mind. If track 2 sounded thin next to track 3 in the cohesion pass, track 2's re-master adds low-shelf weight before the limiter, not after.
6. Second A/B pass against all four re-masters. Write down whether the gaps closed.
