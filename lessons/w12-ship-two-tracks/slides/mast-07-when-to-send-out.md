# When to master vs send to an engineer — a blunt decision tree

The honest question is not "am I good enough?" It is "is this track finished?"

Send it out when:

- You have a label release with a distributor timeline and actual money moving
- The mix is fully committed — bounces, arrangement, mutes, automation, done — and you want a second pair of ears to translate it
- You have four or more tracks for the same release and need loudness consistency across an EP
- The budget is real — $75–$300 per track for an independent engineer, $500+ for a mid-tier house, more for a name

Self-master when:

- You are still exploring — the mix might change
- The release is a Bandcamp private EP to one friend (this week)
- The deadline is shorter than the turnaround of any engineer you would actually hire
- You are twelve weeks into a practice and the learning is the point

An engineer cannot rescue an unfinished mix. A $300 pass on an unfinished track produces a louder version of the same unfinished track, shipped with your name on it. <Citation bib="fridmann_davefridmann_com" />

For this week: self-master. Year Two: decide per release. The decision is situational, not skill-based.
