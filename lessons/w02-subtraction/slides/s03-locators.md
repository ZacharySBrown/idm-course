# Place the locators before you touch a sound

Open Arrangement View. Before you load a single instrument, drop seven locators at these timecodes:

| Locator | Time  | What lives here                       |
|---------|-------|---------------------------------------|
| Intro   | 0:00  | Texture + one constant element        |
| A       | 0:45  | Texture + drums                       |
| B       | 2:00  | Texture + drums + bass                |
| Reduction | 3:15 | Texture + a single vocal chop         |
| Peak    | 3:45  | All elements, densest state           |
| Outro   | 5:00  | Inverse of Intro                      |
| End     | 5:45  | Stop                                  |

Right-click the ruler → Add Locator, then rename inline. Locator names follow the project into bounces.

**What this buys you.** A finite session. The locators are suggestions, not contracts — the act of placing them converts *open-ended* into *shaped*. Zak Handbook §5.2. `bib:ableton_live_12_manual`, Arrangement View → Locators.
