# Four Tet, *Rounds* (2003) — the constant anchor in practice

"Unspoken," the fourteen-minute closer on *Rounds*, is the reference recording for the <term key="constant_anchor">constant anchor</term> idea. A single drum loop rides through the entire track. Everything else — pads, melodica, chopped vocal, live percussion overdubs — mutates around it.

**What to listen for.** The loop never stops. It gets dirtier, it gets ducked under other material, but it never drops out. Hebden's thesis in `Tape Notes #140`: *"Decide which things ride through the track in a simple way, and which the detail is in."* The drums ride. The detail is everywhere else. `bib:tape_notes_140_four_tet`, `bib:hebden_unspoken_production`.

**Why *Rounds* and not a newer record.** Because the constraint was tighter — fewer plugins, fewer options, an anchor-first composer working in a smaller room. The method is more visible in the early work.
