# Mute backward: Reduction, Breakdown, Intro

Duplicate the Peak across the full six-minute timeline. Then, inside each locator region, toggle tracks off with the **Track Activator** (the green square in the track header).

**Default mute map — start here, mutate later:**

- **Peak** (3:45–5:00): nothing muted.
- **B** (2:00–3:15): mute perc 2 + stutter.
- **A** (0:45–2:00): also mute vocal chop + lead.
- **Reduction** (3:15–3:45): mute drums + bass.
- **Intro** (0:00–0:45): only anchor + texture remain.
- **Outro** (5:00–5:45): inverse of Intro.

**The load-bearing trick.** If you lose the kick at bar 24, you don't replace it — you let the Fridmann bass carry the weight, and the listener doesn't notice the trade. Zak Handbook §5.3.
