# The constant anchor — Hebden on what rides through

`Tape Notes #140` — Kieran Hebden:

> *"Decide which things ride through the track in a simple way, and which the detail is in."*

**Translation into Subtraction.** Pick one element in the Peak that never mutes for the whole six minutes. Candidates:

- A single pad note
- A hat pattern running 1/16
- A sub drone on the root
- A field-recording bed

That element is the **constant anchor**. Not interesting on its own — that's the job. It gives the listener a thread through the chaos while everything else mutates.

**Deflation.** An anchor will not save a bad Peak. If the densest state is thin, the anchor exposes the thinness across the whole track. Finishing move, not a rescue move. Handbook §5.8.
