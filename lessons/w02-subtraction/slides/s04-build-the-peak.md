# Build the Peak first — the densest sixteen bars

Write the climax of the track first. Sixteen bars. Everything on.

**Checklist for the Peak loop:**

- Kick, snare, hats, one perc layer
- Sub + mid bass (two tracks, separated bands)
- One melodic element (pad or lead, pick one)
- One texture (field, noise bed, reverb wash on a send)
- One vocal chop or one-shot
- One "detail" element — a ratchet, a click, a stutter

**The constraint.** If it is not in the Peak, it does not exist in the track. You cannot add a new element at bar 96 because the section feels thin. Thin is the point — Subtraction doing its job.

Loop the sixteen bars in Session View. A/B against a reference. When you enjoy it, stop and bounce. Zak Handbook §5.3, §6.1.
