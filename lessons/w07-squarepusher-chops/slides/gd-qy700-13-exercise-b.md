# Exercise B — explore beyond

Four-instance Rhythm-generator patchwork — Jenkinson's multitimbral posture in modern form.

1. Four MIDI tracks. Each loads **Rhythm** → **Velocity Shaper** → **Simpler** (or Operator). <Citation bib="ableton_live_12_manual" />
2. Assign a different seed to each Rhythm instance: 17, 42, 99, 203. Do not choose them — type them blind.
3. Each track's Simpler gets a different sample — one kick, one snare chop, one FM-ish pluck (Operator preset), one percussion hit.
4. Each Velocity Shaper gets a different hand-drawn curve. Track 1 ramps up; track 2 ramps down; track 3 is flat with random spikes; track 4 is two steps loud, six steps quiet.
5. Clip length: 2 bars each, all free-running. Loop them against each other. The collisions are the composition.
6. Bounce a 32-bar arrangement. No quantize fixes. No re-seeding.

The output should feel like four different players reading four different sheets of music in the same room. That is the QY700 posture. The seeds are the blind numeric entry.
