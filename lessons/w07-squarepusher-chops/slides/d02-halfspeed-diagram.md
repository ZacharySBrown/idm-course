# Half-speed tape bass — diagram

The half-speed trick, visualized.

```
[ STAGE 1 — record at half speed ]
project tempo:    160 BPM         →  set to 80 BPM
bass part:        written in key  →  played an octave DOWN
tape/sequencer:   runs at 7½ IPS  →  (tempo halved, pitch halved)
result on disk:   slow, low bass, tracking errors DIMINISHED

[ STAGE 2 — return to speed ]
project tempo:    80 BPM          →  back to 160 BPM
warp mode:        Complex Pro + Formants ON
pitch:            snaps up one octave to original
timing:           was loose at 80, becomes TIGHT at 160
```

The math: timing error of ±10 ms at the tracking stage (80 BPM) becomes ±5 ms at playback (160 BPM). Human-player jitter is halved. <Citation bib="tape_op_89" />

The "bionic swing" Jenkinson describes is this halving applied to a player who already has feel. You are not quantizing; you are compressing time around the player's natural groove. It keeps the swing signature while killing the errors that would expose it as played-in.
