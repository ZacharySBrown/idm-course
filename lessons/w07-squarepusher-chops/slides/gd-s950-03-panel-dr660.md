# Annotated tour (2/2) — DR-660 pattern grid + sync

The DR-660 was a budget drum box that Jenkinson hotwired into a trigger sequencer for the S950.

- **Pattern programming.** Step mode or real-time. Jenkinson used real-time tap-in for the feel, then re-quantized to 1/16 or 1/32.
- **Trigger Out (MIDI).** Each pad sends MIDI note-on on a configurable channel. Route the DR-660's MIDI out to the S950's MIDI in; each pad fires a sample in a keygroup.
- **Sync Out.** 24 PPQ clock — the tape-era standard. Slaves the S950's internal LFOs and the QY700 when used alongside.
- **Accent row.** A second-level velocity lane. Three velocity tiers only; Jenkinson used the accent row as a ghost/loud switch, not as nuance.

Limitation as feature: the DR-660 has no shuffle beyond a single global swing. The groove has to come from the pattern itself.
