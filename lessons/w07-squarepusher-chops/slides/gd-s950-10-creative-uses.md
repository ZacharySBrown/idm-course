# What else can you do with the S950 chain

Beyond the Amen:

- **Held-sample stretching.** The S950's pitch-shift aliases; deliberately set the loop tight and run the pitch wheel on the DR-660 to create a sustained, pitched noise bed — the *Hard Normal Daddy* snare rolls were generated this way.
- **Reversed+pitched layers.** Reverse a kick, repitch up 5 semitones, layer under the dry kick on a slight delay. The attack becomes a swell.
- **Inter-keygroup modulation.** The S950 has per-keygroup filter envelopes. Assign two keygroups to the same pad via velocity ranges — low velocity triggers one, high velocity triggers the other. Effectively a per-pad round-robin.
- **Resample the S950 output back to tape.** Jenkinson did this to flatten the dynamics and print the companding artifacts into a fixed source — then re-sampled *that* back into the S950 for a double-companded texture. The Metallurgist era is built on this.

All of it translates to Drum Rack + Follow Actions + resample-to-clip.
