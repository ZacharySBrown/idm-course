# Yamaha QY700 — the no-computer sequencer

Tom Jenkinson's brain on *Go Plastic* is a thirteen-pound grey plastic box with a 320×240 monochrome LCD and a numeric keypad. That is the <term key="qy700">Yamaha QY700</term>, released 1996. <Citation bib="tape_op_89" />

Sixteen-part multitimbral <term key="awm_synthesis">AWM</term> tone generator, 32-track sequencer, 480 PPQN resolution, 48-voice polyphony, step-entry-first interface. Four MIDI outs. No monitor. No mouse.

Jenkinson, Tape Op #89: *"In order to really work fluently in a studio you need to have a model, or virtual image, of all those pathways in your head."* The QY700 forces that. You cannot see the signal path — you have to carry it.

The rest of this module is an annotated tour.
