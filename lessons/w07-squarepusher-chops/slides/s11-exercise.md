# Ship the chop — one pass, no undo

**Deliverable.** A 30-second stereo bounce of an Amen-style rearrangement built from the `w07-funky-fanfare-amen` drums render.

## Steps

1. Drop `build/audio/stemforge-renders/w07/ff_drums.wav` onto an audio track. <Citation bib="stemforge_repo" />
2. Right-click → **Slice to New MIDI Track** → Slice to: **1/16**, Slicing Preset: **Built-In**. <Citation bib="ableton_live_12_manual" />
3. On the resulting MIDI track, arm record. Set a 2-bar loop at project tempo.
4. **Single pass.** Finger-drum or keyboard-play a new pattern across the Drum Rack chains. No undo. No piano-roll editing after the take.
5. Bounce-in-place the MIDI clip to audio. Delete the source MIDI. Keep the audio. <Citation bib="ableton_live_12_manual" />
6. Trim to 30 seconds. Export as `w07_<yourname>_single-pass.wav` (24-bit, 44.1 kHz, stereo).

## Constraint

If a pass is unusable — actually unusable, not just uncomfortable — you get exactly one re-pass. That is the budget. Ship the second pass even if it is worse.

Include a one-line note: which slice you hit the most and why. Hand in via the course Dropbox.
