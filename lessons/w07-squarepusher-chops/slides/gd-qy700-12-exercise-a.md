# Exercise A — replicate the technique

Program a 32-step Squarepusher-style Amen fragment using **only** Ableton's step-input mode. No real-time play-in. No piano-roll draw.

1. New MIDI track → load Drum Rack with the stemforge Funky Fanfare chops from `build/audio/stemforge-renders/w07/ff_chops/`. <Citation bib="stemforge_repo" />
2. Empty MIDI clip, 2 bars. Push into step-input mode (Edit → Draw Mode + step pointer, or Push 3 step mode if you have hardware).
3. Enter 32 steps of pitches by typing notes on the keyboard with the clip in step-input. Each keystroke advances the pointer by 1/16. <Citation bib="ableton_live_12_manual" />
4. Do **not** open the piano roll to drag anything after the fact.
5. Walk the velocity lane and hand-draw per-step values — top row kicks loud, mid-row snares medium, hat/ghost hits quiet.

Bounce the clip. Name it `w07_qy700-style_<yourname>.wav`. One pass. If you make a mistake mid-entry, keep going — the mistake is the take.
