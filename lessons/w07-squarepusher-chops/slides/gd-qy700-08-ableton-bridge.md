# Ableton bridge — Push step mode + MIDI generators

Ableton's answer to the QY700's step grid is the <term key="push_step_sequencer">Push 3 step sequencer</term> plus Live 12's native MIDI generator devices. <Citation bib="ableton_live_12_manual" />

Parameter mapping:

| QY700 feature | Ableton equivalent | Notes |
|---|---|---|
| Step grid | Push step mode | 16 pads per bar, 1/16 default, shift for finer resolution |
| Velocity editor | Push per-step velocity + Velocity Shaper MIDI device | Per-step values or a curve across selection |
| Pattern chain | Session view scene chain + Follow Actions | Scenes = patterns; Follow Actions = chain logic |
| 16-part multitimbral | 16 MIDI tracks each with its own instrument | External Instrument device for hardware MIDI outs |
| 480 PPQN internal | Live is 24 PPQN clock, sub-tick timing via sample-accurate engine | Ableton is finer than QY700 when bounced in-place |

The screen problem does not disappear. Push mitigates it — step entry on the pads, eyes down on hardware — but you still have a laptop behind you. Turn it away.
