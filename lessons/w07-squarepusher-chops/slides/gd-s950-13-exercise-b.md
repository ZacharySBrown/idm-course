# Exercise B — Metallurgist / Hello Everything territory

Reverse half the chops + shuffle them + bitcrush. Creates the synthetic-drum terrain of *The Metallurgist* (2001) and *Hello Everything* (2006).

1. Start from the Drum Rack built in Exercise A.
2. For every second pad (C1, D1, F#1, A1 — any half of the rack), open the Simpler and toggle **Reverse** on.
3. Insert an **M4L Buffer Shuffler 2.0** before the Drum Rack. Set freeze length 1 bar, divisions 1/16, random mode ON.
4. Shift the bitcrush harder — Redux Bit Depth 8, add a Saturator (Type: Soft Sine, Drive 6 dB) after the bitcrusher.
5. Play the same 32-bar pattern from Exercise A through the modified rack.
6. Capture MIDI. Freeze + Flatten the Shuffler so the re-sequence is printed.

The output should not sound like the Amen. It should sound like someone sampled an Amen, lost it, and reconstructed it from memory. That is the Metallurgist posture.
