# Go Plastic — the rig, receipts

The *Go Plastic* (2001) rig, verbatim from Tape Op #89: *"I didn't use a computer on Go Plastic. It was made with a Yamaha QY700, TX81Z and FS1R, an Eventide DSP4000 and Orville, an Akai S6000 and a Mackie 16-channel desk."* <Citation bib="tape_op_89" />

Signal path, named:

- **Yamaha QY700** — sequencer and MIDI brain.
- **Yamaha TX81Z** (1987) + **Yamaha FS1R** (1998) — the FM voices. Metallic, brittle, lopsided by design.
- **Akai S6000** — 128 MB sampler, the Amen lives here on *Go Plastic*-era material (the earlier S950 era precedes it).
- **Eventide DSP4000 + Orville** — the effect hub. Running custom algorithms Jenkinson wrote in Eventide's **VSIG** modular-patch environment on a PC, transferred via SysEx.
- **Mackie 16-channel desk** — sum. Nothing exotic. The character is upstream.

Specific patch, same interview: *"The bass distortion on 'Megazine' was done with an old-school 110V Morley Wah pedal and an Orville distortion patch based on a curve, X/Y-mapping module."*

We do not need the hardware. We need the mental model.
