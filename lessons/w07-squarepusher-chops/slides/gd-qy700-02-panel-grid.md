# Annotated tour (1/2) — the step grid

The QY700's front panel is organized around the <term key="step_sequencer">step sequencer grid</term>. Each track is a row; each column is a 1/16 (or finer, up to 1/96 at 480 PPQN). Jenkinson enters notes by typing pitch + velocity + gate-time as numbers. <Citation bib="tape_op_89" />

Four controls that matter for the technique:

- **Step grid.** One line per track, up to 32 tracks. No piano-roll view. You read pitches as hex/decimal numbers in columns.
- **Velocity editor.** Per-step velocity curves. This is where Jenkinson "humanized" patterns — manually, step by step, not via groove template.
- **Pattern chain.** Up to 16 patterns chained into a song. Each pattern is a loop; the chain is the arrangement.
- **Part select 1–16.** Sixteen AWM voices, each on its own MIDI channel, addressable from any track.

You can see why the screen was the enemy. The grid *is* the score.
