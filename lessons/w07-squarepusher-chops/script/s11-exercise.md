# s11 — Exercise

Exercise. [pause 400ms]

Deliverable: a thirty-second stereo bounce of an Amen-style rearrangement built from the `w07-funky-fanfare-amen` drums render. File named `w07_<yourname>_single-pass.wav`. Twenty-four bit, forty-four-point-one kilohertz, stereo. Hand in via the course Dropbox.

Steps. [pause 200ms]

One. Drop `build/audio/stemforge-renders/w07/ff_drums.wav` onto an audio track in a new Ableton project. Set the project tempo to whatever matches — Funky Fanfare is roughly one-oh-two BPM as rendered.

Two. Right-click the clip. Slice to New MIDI Track. Slice to: one-sixteenth. Slicing Preset: Built-In. Hit Create. You now have a Drum Rack with sixteen chains per bar — one per sixteenth-note slice. [pause 250ms]

Three. On the new MIDI track, arm record. Set a two-bar loop at project tempo. Set the count-in to one bar so you are in the pocket by the time recording starts.

Four. **Single pass.** [pause 300ms] Finger-drum a new pattern across the Drum Rack chains. Or keyboard-play it if you do not have a pad controller. Do not plan the pattern. Do not sketch it in the piano roll first. Play it once. Whatever happens, happens.

No undo. No piano-roll editing after the take. No quantize. No velocity shaping. No ornament. This is a raw pass.

Five. Bounce-in-place the MIDI clip to audio. Right-click the clip, choose Bounce. Delete the source MIDI. Keep only the printed audio. ~Recall-hostile. Committed.~ [pause 200ms]

Six. Trim to thirty seconds. Export.

The constraint. [pause 300ms] If a pass is genuinely unusable — meaning there were hardware problems, or you panicked and hit nothing, or the tempo was wrong — you get exactly one re-pass. That is the budget. If the second pass is worse than the first, ship the second pass anyway. The budget does not renew.

Include a one-line note in the hand-in. Which slice did you hit the most, and why? If you do not know why, say you do not know why. The honesty is the assignment.

One last thing. If you finish this in under twenty minutes, you are doing it right. This is not a session where you build a full track. This is a session where you make a piece of raw material you could build a track *out of*. Next week's lesson uses material like this. Week ten's lesson uses it more aggressively.

Go. [pause 300ms]
