# Ableton bridge — Push 3 step mode + per-step automation

The Elektron p-lock paradigm maps cleanly onto two Ableton 12 surfaces.

**Push 3 step edit mode.** Per the Push 3 manual: hold a step pad, turn any device knob — that parameter value is locked to that step only. Same gesture as FUNC + step on the Machinedrum. Push also exposes the Rhythm generator Drum Rack layout for live Auto-generation per voice.

**Clip envelopes per parameter.** On a MIDI or audio clip, expose an envelope for any device parameter (filter cutoff, FM ratio, send level). Draw 16 breakpoints across 16 steps. The clip now carries per-step parameter data just like a Machinedrum pattern.

| Machinedrum concept | Ableton 12 equivalent |
|---|---|
| Per-step p-lock | Push 3 step-edit, or clip envelope breakpoint |
| Per-track pattern length | Clip loop length (set per-clip independently) |
| Kit snapshot | Chain Selector preset in a Rack |
| MIDI sync between machines | Any MIDI track routed to another's input |
| Machine type swap | Rack Macro Variation, or an Instrument Rack chain select |
