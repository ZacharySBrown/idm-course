# Parallel modern workflows

Max has competitors. Some share DNA; some take the idea in different directions.

- **Pure Data (Pd)** — the open-source cousin, by the same person (Miller Puckette) who wrote Max's predecessor at IRCAM. Visually similar; fewer high-level abstractions; runs on a Raspberry Pi.
- **Reaktor Core (Native Instruments)** — similar graph-based DSP, different object library, shipped with Komplete.
- **VCV Rack** — modular-synth-as-software. Modules are wired like Max signal objects. Much narrower scope.
- **Bitwig Studio Grid** — built into Bitwig, audio-rate + modulation graph. The most elegant modern take on the signal-flow idea.
- **SuperCollider** — text-based, client-server, has the longest research pedigree. Not visual; closest to academic computer music.
- **TouchDesigner** — Jitter's video-side competitor. Sound is a second-class citizen; visuals are first-class.

Autechre have worked in several of these at different times. Booth has mentioned SuperCollider and custom C code on WATMM. The point: **the paradigm is transferable.** Learn the dataflow model once, port it to whichever tool the session demands.
