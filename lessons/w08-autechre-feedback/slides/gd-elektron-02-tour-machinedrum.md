# Machinedrum tour — the five controls that matter

Six-track pattern sequencer (16 steps, 16 patterns × 16 banks = 256 patterns), two global effects, and **per-step parameter locks** on every control visible on the front panel.

What to look at on the panel:

- **Trig keys (16).** Per-step note on/off. Classic Roland-style entry.
- **Function + Trig.** Hold FUNC, hit a trig — you arm a <term key="plock">p-lock</term> on that step. Whatever knob you move next is now locked to that step only.
- **Param encoders (6 rows × 8 columns).** Synthesis parameters per machine. On the UW model: SYN, AMP, FLT, EFF, LFO, ROUT — each 8 knobs.
- **Kit banks.** 64 kits per project. A kit is a machine-type + parameter snapshot per track.
- **Machine selector.** TRX (analog modelling), EFM (FM drums), E12 (12-bit sample playback), P-I (physical impulse), GND (noise-based). Swapping machine type on a track is a single button press.
