# Exercise A — the 16-pitch snare

Replicate the Machinedrum p-lock move in Ableton. No Elektron hardware required.

Steps:

1. Open Push 3, create a new MIDI track, load Drum Rack with one slot — slot D1 is a snare Simpler loaded with any crisp rim/snap sample.
2. Push: step mode. Program trigs on steps 1, 3, 5, 7, 9, 11, 13, 15 — every odd step.
3. For each trigged step: hold the step pad, turn the Simpler **Transpose** knob to a different semitone value. Write down the 8 values.
4. No randomization. No dice. You are picking eight pitches by hand.
5. Loop, listen. Commit the MIDI clip (right-click → Freeze + Flatten the track if you want the audio).

**Deliverable.** One 2-bar bounced snare pattern with 8 deliberate pitches. Nothing else — no reverb, no effects. The point is the p-lock discipline in isolation.

Target time: 15 minutes.
