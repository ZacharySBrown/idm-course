# Parallel use cases in modern workflows

The Elektron workflow survived the decade it defined because the constraint set is still useful. You can buy a new Machinedrum spiritual successor — **Digitakt II, Digitone II, Analog Rytm MKII** — all ship with the same p-lock grammar.

In a computer workflow the same posture maps to:

- **Push 3 standalone.** A battery-powered Elektron equivalent in Ableton's house style. Step-edit on Push does the p-lock gesture natively.
- **Bitwig Studio's modulators + grid.** Any parameter can be modulated per-step by a discrete sequencer modulator. No Elektron box required.
- **Renoise tracker.** The 90s/00s tracker UI already had per-step effect columns — literally a p-lock for every FX slot. The cultural ancestor.
- **Novation Circuit Tracks/Rhythm.** Cheap modern implementation of the Elektron grammar.

The transferable skill is not the hardware. It is the discipline of **deciding one parameter per step on purpose**. That discipline ports.
