# Monomachine tour — six voices, six engines

Six-track mono synthesizer. Same step-sequencer architecture as the Machinedrum, same p-lock discipline, different sound-generation backend.

The machine types you can load per track:

- **SID** — emulates the Commodore 64 6581 chip. Square/pulse/tri, ring mod, hard sync. Gritty.
- **FM** — 4-operator FM. DX100-adjacent. The Autechre metallic-bell timbre lives here.
- **WAVE** — single-cycle wavetable. 16 banks of 64 waves.
- **VO** — formant-synthesis vocal-ish. Bizarre.
- **DPRO** — digital pulse. Cheap PWM with modulation.
- **SuperWave** — analog-modelling supersaw.

Per-track effects: AM, distortion, filter (2-pole LP/BP/HP/BS), delay, reverb send. All p-lockable.

The live interplay: Monomachine sequences melodic material; Machinedrum sequences drums; Nord G2 routes audio-rate modulation between them via CV; both Elektrons sync to the same MIDI clock.
