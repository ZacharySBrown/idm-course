# Euclidean + unlinked clip envelopes = polymeter

Two tricks, stacked.

**Euclidean generator** (Live 12 manual): distribute N pulses across M steps with rotation R. The Ali Jamieson method:

- Kick 16/4/0
- Snare 16/3/2
- Hat 16/7/1
- Clave 12/5/0

Four clips, unequal lengths, one Drum Rack. Pattern repeats only at the least common multiple. Forty-plus bars before anything literally recurs.

**Clip envelopes unlinked from loop length.** In a clip envelope, disable "Linked" on, say, the filter cutoff. Envelope length 7 bars. Audio clip length 4 bars. The filter sweep never lines up with the drum cycle the same way twice.

- 4-bar drum audio + 7-bar filter LFO = the Autechre polymeter posture.
- 3-bar pan + 5-bar amp envelope = fully non-repeating motion on a pad.

No randomness touched. Every moment is procedurally derivable. This is the point.
