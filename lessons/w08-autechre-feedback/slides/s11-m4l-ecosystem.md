# Max for Live — chaos, Markov, probability

Three devices. Each replaces a hand-built Max/MSP structure the Autechre duo would have patched themselves.

- **K-Devices ESQ / HEXO.** Per-step probability step sequencers. Each step carries its own chance value. ESQ is melodic; HEXO is percussive. This is the *"we may have one fader that determines how often a snare does a little roll"* posture from the SOS 2004 piece, ported to a UI.
- **Dillon Bastan — Strange Mod.** 3D chaotic attractors (Lorenz, Rössler, Thomas). Navier-Stokes-adjacent. Map X/Y/Z outputs to filter, pan, pitch. This is the *Confield* modulator, 2026-edition.
- **Markov Variations.** Markov-chain note sequencer. Feed it a source melody, it generates statistically consistent derivatives. Same family as the Oversteps-era voice-writing patches Booth has described on WATMM.

Pick **one.** Add the others after you ship a track with the first. Tool-collecting is the trap *Draft 7.30* warned us about.
