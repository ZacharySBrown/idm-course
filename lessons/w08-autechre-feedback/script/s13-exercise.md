The exercise. [pause 400ms]

Deliverable: a thirty-two-bar bounced audio file of evolving-but-rule-based drums. One file. Exported. Named and dated.

Step one. Load the stemforge render for this week — `w08-moanin-ambient-feedback`. The wet file, `moanin_ambient.wav`. Drop it into a new audio track in Live. Right-click the clip and select Slice to New MIDI Track. Slice by Transient. Set the resulting Drum Rack as your kit.

Step two. New empty MIDI clip, eight bars, on that Drum Rack's track. Apply Rhythm generator — Steps sixteen, Density fifty-five percent, Division one-sixteenth. [pause 300ms] *Commit.* Duplicate the clip to a scratch track named `rhythm_v01`.

Step three. Apply Recombine in Shuffle mode. Commit. Scratch duplicate as `recombine_v01`.

Step four. Apply Time Warp with a breakpoint curve — one-point-five times at bar one, drop to zero-point-five times at bar four, return to one-point-zero at bar eight. Commit. Scratch duplicate.

Step five. Apply Velocity Shaper — draw a hand curve, no preset. Commit. Then apply Ornament at Length one-sixty-fourth, Density twenty percent. Commit. That is the four commits — five if you split Velocity Shaper and Ornament, and I do. [pause 400ms]

Step six. Loop the eight-bar clip four times in Arrangement view to get thirty-two bars. Open the audio clip's filter envelope. *Unlink* it. Set envelope length to seven bars. The filter sweep now polymeters against the drum loop.

Step seven. Insert Roar on the Drum Rack's output. MultiBand mode. Use the settings from slide nine — Tube low, Diode mid, Shards high, Feedback at fifteen percent, one-sixteenth time.

Step eight. Freeze the track. Flatten. Export thirty-two bars as WAV. File name — `w08_autechre_chain_v01.wav`. Save the set as `w08_v01.als`. Delete the source MIDI.

Time-box: forty-five minutes. [pause 500ms] Ship one bounce. Stop. [pause 300ms] That is the week.
