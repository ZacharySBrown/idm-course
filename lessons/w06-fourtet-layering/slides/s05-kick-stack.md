## Daydream Repeat — the four-layer kick stack

`Tape Notes #140`. Hebden walks through "Daydream Repeat" element by element. Four layers, each one job.

1. **Main kick sample** — the recognizable thump. Midrange body, character. A drum-machine sample that already felt right.
2. **Sine sub** — tuned to track root. On headphones "almost inaudible… but just gives the weight in a big club." Operator sine, short envelope.
3. **Transient click** — high-frequency click above the main attack, for small-speaker cut-through. Band-passed above 3 kHz, 10 ms envelope.
4. **Sub body** — longer, slower sustain for weight past the transient. Low-passed sine tail at 120 Hz, 200–300 ms release.

Process: added iteratively over months of club playbacks. Each layer solved a problem that only surfaced on a large PA. `Tape Notes #140`.

Rule: solo each, confirm one job, sum.
