## Ableton bridge — Effect Rack with Chain Selector

Closest native analog to AudioMulch's Metasurface morph.

- **Audio Effect Rack** — multiple chains, each chain is an independent plugin sequence.
- **Chain Selector** — a single knob/macro that crossfades between chains as it sweeps. Maps chain-position to signal routing.
- **Macro Controls** (8 macros per rack) — each can be mapped to parameters across any chain. Map them to filter cutoffs, delay times, reverb sizes — and a single macro sweep morphs the whole processing character.

| AudioMulch concept | Ableton equivalent |
|---|---|
| Contraption graph | Effect Rack chain sequence |
| Metasurface | Chain Selector + multi-assigned Macro |
| Live parameter morph | Record automation on macros while playback runs |
| Print to disk | Render selected track, reimport |

This is the bridge. Not identical, but the gesture — *morph a processing chain across a 16-bar print* — is achievable.
