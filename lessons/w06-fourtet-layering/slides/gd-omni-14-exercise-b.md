## Exercise B — explore: four-layer kick from Simpler + Operator only

**Deliverable.** Daydream Repeat's four-layer kick stack (see `s05`) recreated using only **Ableton Simpler + Operator** — no third-party, no factory Drum Rack kit.

**Steps.**

1. **Main kick** — Simpler, Classic mode, any one-shot kick sample, pitched to F1.
2. **Sub sine** — Operator, single sine op, Ratio 0.5, release 120 ms, tuned to F1.
3. **Transient click** — Simpler with a noise/hat one-shot, band-passed 3–6 kHz, envelope 10 ms.
4. **Sub body** — Operator, sine op, Ratio 0.25, release 300 ms, LP at 120 Hz.

Group the four tracks. Trigger all four off one MIDI clip via MIDI Routing or Clip send.

**Constraint.** No EQ beyond the click's band-pass. No compression. If the stack does not feel heavy, rebalance with track volume, not with plugins. This is the Hebden stack logic — layers are chosen to not fight, not EQed to not fight. `Tape Notes #140`.
