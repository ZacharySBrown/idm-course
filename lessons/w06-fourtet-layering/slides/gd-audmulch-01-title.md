## AudioMulch + Cool Edit Pro — the *Rounds* chain

Rewind to 2003. The Hebden in this module is not a preset browser. He is a sample mangler working out of a North London flat on Windows XP.

- **AudioMulch** — modular live-audio patching host built by Ross Bencina, Australia, Windows only for most of its life. Peak IDM adoption ~2001–2006. `bib:audiomulch_docs`.
- **Cool Edit Pro** — Syntrillium's destructive waveform editor. Adobe acquired it in 2003 and shipped the code as Audition 1.0. `bib:cool_edit_pro_legacy`.
- Estimated *Rounds* source samples per album: **200–300**, per-track **20–30**. Zak Handbook synthesis — flagged in `needs_bib`, not a confirmed primary interview number.

Two dead tools. One alive discipline. The logic is worth excavating.
