## Annotated tour 5/5 — signal path in one line

Soundsource(s) + synth osc → per-layer filter → per-layer envelopes → Layer FX → Part filter → Part FX → Aux FX → master out.

Most of those stages are bypassed on the default patch. The sounds you hear arrive via the Soundsource + Layer FX path. That is where the factory character lives.

For Hebden, only two of those stages matter at session time:

1. **Which patch** — the whole top-of-chain decision.
2. **Orb / mod-wheel** — if anything, live expression only.

The filter page, the envelope page, the oscillator edit view — unvisited. The whole box has been pre-patched by Spectrasonics programmers for him to browse past.
