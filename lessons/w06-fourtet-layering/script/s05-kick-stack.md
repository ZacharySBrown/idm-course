The kick stack. `Tape Notes #140`, mid-episode, Hebden walks through "Daydream Repeat" element by element. Four layers, each doing exactly one job. This is the case study of the whole lesson.

[pause 400ms]

Layer one — the main kick sample. The recognizable thump. The body you would describe if someone asked you what the kick sounds like. He does not say which sample, but the description fits a drum-machine kick at moderate length, processed *exactly as much as it needs to be and no more*. This layer is the character.

[pause 400ms]

Layer two — a sine sub. Tuned. Root of the track. Hebden's phrasing is precise — *"almost inaudible on headphones, but just gives the weight in a big club."* On a MacBook speaker, the sine is functionally gone. On a pair of Sennheiser HD25s it is a faint pressure. On a twelve-foot sub in a club, it is load-bearing.

[pause 500ms]

For our purposes: an Operator sine, Coarse zero, Ratio 0.5 for the octave below the main kick's fundamental, short envelope, no modulation.

[pause 400ms]

Layer three — a short transient click. A high-frequency pop sitting on top of the main kick attack. Hebden's word is *"cutting through."* It exists for the small-speaker problem — when the sub is gone and the body is compressed, the click is what remains. Think band-passed noise burst, three to six kilohertz, ten-millisecond envelope. Not a hat. Not a snare. A click.

[pause 400ms]

Layer four — the sub body. A second, *longer* sub tail. This is where the weight lives past the transient. Slightly lower than layer two, or the same pitch but with a longer release. Two to three hundred milliseconds. Low-passed at 120 hertz.

[pause 500ms]

Process, direct quote-adjacent from the episode — he added these layers *iteratively* over months of playing the track out in clubs. Each layer answered a specific failure mode on a specific rig. Not designed in a session. *Tuned* across a touring calendar.

[pause 400ms]

The rule we take home. Solo each layer. Confirm it does one job. Then sum. If you cannot name the one job a layer is doing in a single sentence, delete the layer. It is not earning its headroom.
