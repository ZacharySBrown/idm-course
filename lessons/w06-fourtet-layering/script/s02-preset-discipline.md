Start with the receipt. `Tape Notes #140`, 2024, host Jon Tipler in conversation with Kieran Hebden about the album *Three*.

[pause 400ms]

Hebden names his synth palette and it is one plugin. Spectrasonics Omnisphere. Quote: *"the synth, main synth plugin I use for absolutely everything."* He browses the preset library *alphabetically*. He plays one called "Plectra Sub" and another called "Swaggering Around." He does not modulate. He does not detune. He does not re-layer oscillators.

[pause 400ms]

For orchestral material — and there is a harp on "Daydream Repeat" — he uses Native Instruments Kontakt with a factory orchestral patch. For width, SoundToys MicroShift. On the master bus, FabFilter Pro-L2 into iZotope Ozone Maximizer. That is the chain. There is no *secret* third plugin. There is no analog console.

[pause 300ms]

Hardware exists — there is a Roland TR-8, an Arturia DrumBrute, two cassette four-tracks he leaned on hard during lockdown — but hardware is a sample-making loop, not a production loop. His own words, from an earlier interview: *"I'll spend time with hardware and instruments to create samples. I'll put the drum machine away and never use it again."* Patch it. Capture a few loops. Put it away.

[pause 400ms]

The posture is the lesson. When Hebden sits down to write, the question *what synth should I use* has already been answered months ago. Omnisphere. The only remaining question is *which preset, in alphabetical order, gets me going*. That takes thirty seconds. The rest of the session is arrangement.

[pause 500ms]

Compare that to the default modern-producer session — forty-five minutes in Serum tweaking a bass patch before you have written a note. That is the rabbit hole this constraint is designed to block.

[pause 400ms]

Rule of thumb, taken from the discipline: *pick a preset library, pick it for the whole track, do not shop again mid-session*. If you catch yourself browsing a second library, close the browser and go back to the clip.
