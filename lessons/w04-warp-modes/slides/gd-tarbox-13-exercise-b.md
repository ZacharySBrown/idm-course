# Exercise B — explore — broadcast multiband beyond bass

The Dorrough 610 was not designed for bass guitar. It was designed to make FM radio loud without clipping the transmitter. Keep the Rack you built in Exercise A. Put it on three things it was not intended for, and listen.

1. **Drum bus.** The Rack on a stereo drum bus approximates New York mastering style — aggressive multiband crushing for loudness. Settings from Exercise A will be too aggressive; back the low-band threshold up to −20 dB, ratio 4:1. Bounce.
2. **Full mix.** Same Rack, insert before your mastering chain. This is what FM broadcast multiband actually does for a living. Bounce.
3. **Vocal.** Use it on a vocal take. The mid band will become brutal; this is how <term key="mid_side">mid-side</term> mastering tricks make a voice sit forward. Bounce.

Write one paragraph — not a bullet list — explaining what broadcast multiband does when it is *not* abused on purpose. The answer is boring and useful: it equalises perceived loudness across the spectrum so the transmitter runs hotter without any one band popping the limiter. That is it.
