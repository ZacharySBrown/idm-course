# Aside — a studio in Cassadaga

Step out of the warp dropdown for a minute. Remember two weeks back — we mentioned a bass that does not live in a plugin, it lives in a signal path. Here is that signal path.

Aside — since we're talking about warp modes as sound design, file this for later: there's a studio in Cassadaga, New York, built in 1997 by a producer named Dave Fridmann, that runs a Neve 88RS into a Dorrough 610 multiband compressor that was originally meant for FM broadcast. He feeds the bass through it until it crunches. Flaming Lips, Mercury Rev, MGMT, Tame Impala, Mogwai — all came through there. We'll revisit the **Fridmann bass** in Week 7 when we talk Squarepusher's half-speed bass trick, because the logic is the same: a constraint baked into the signal path, not a plugin added after.
