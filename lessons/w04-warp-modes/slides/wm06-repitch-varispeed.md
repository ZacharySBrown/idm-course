# Re-Pitch — pitch and time locked, no DSP at all

<term key="repitch_warp">Re-Pitch</term> is the odd one in the dropdown because it does no time-stretching. It reads the clip at an altered sample rate. That is it.

Read faster, pitch goes up and duration goes down. Read slower, both go down. This is exactly what an analog tape machine does when you grab the reel and push it — which is why the result is called <term key="varispeed">varispeed</term>. No FFT, no granulation, no algorithmic artifacts. You get the sound of a tape transport running wrong.

This is also the mode that built <term key="chipmunk_soul">chipmunk soul</term> — Kanye's early Roc-A-Fella beats, pitched up four to seven semitones on a soul sample, with the duration contracting by the same ratio. Coupled pitch-and-duration is the whole character.

When to reach for it: any time you *want* the artifact. Any time you *don't*, you want Complex Pro.
