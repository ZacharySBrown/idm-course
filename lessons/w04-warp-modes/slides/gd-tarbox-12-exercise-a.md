# Exercise A — patch the Fridmann bass on a sub sample

Find a sub-bass sample. Anything clean, 8 bars, one note or a simple pattern. The Hallogallo stem does not apply here — the drums-and-guitar split does not include isolated bass.

Deliverables:

1. **Dry bounce.** Sub through a track with no processing. Label `fridmann-bass-01-dry.wav`.
2. **Multiband Rack only.** Build the three-chain Rack from the previous slide. Bounce. `fridmann-bass-02-rack.wav`.
3. **Rack + Saturator.** Add the Saturator Analog Clip after the Rack. Bounce. `fridmann-bass-03-rack-sat.wav`.
4. **Optional — Rack + Decapitator Style N.** If you own it. `fridmann-bass-04-decap.wav`.

Write three sentences. One per stage (dry → Rack → Rack+Saturator). Each sentence should name what the stage added or removed: low-band compression artifact, harmonic thickness, top-end dulling.

Zip as `w04-fridmann-bass-AB.zip`. 30 to 45 minutes. Do not spend 90.
