# Plugin shortcut — Decapitator Style N (Neve germanium)

If you own <term key="decapitator">SoundToys Decapitator</term>, there is a faster route. <term key="decapitator_style_n">Style N</term> models a Neve 1057 germanium stage — not the 88RS, but same manufacturer, same era of non-linearity. Per the compass artifact's Decapitator decoder (`bib:soundtoys_decapitator_manual`):

- **Style N** — dull/weighty, thickens synths and bass
- **Drive 7** — deep into the modelled non-linearity
- **Punish on** — engages an extra compression stage inside the plugin
- **Low-Cut halfway** — keeps DC and mud out of the saturator
- **High-Cut ~6 kHz Steep on** — drops the high-band character the way a Neve channel would

Drop Decapitator Style N *after* the multiband Rack. The plugin handles both the "hot Neve pre" role and the "tape thickening" role in one stage. Less surgical than separate Saturators, more authentic character. Pick per mix.

Caveat: the Style N model is of a Neve *input stage*, not a broadcast multiband compressor. It gets you the harmonic-character half of the Fridmann bass — it does not replace the multiband part. Keep the Rack upstream.
