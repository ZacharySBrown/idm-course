# Settings to remember, pitfalls to flag

Compact reference. Commit these to muscle memory.

**Complex Pro — presets**

- Vocals, pitched up: Formants 100, Envelope 64
- Full mix, moderate stretch: Formants 100, Envelope 128
- Sub bass, long stretch: Formants 100, Envelope 256
- Aliased chipmunk character: Formants 0, Envelope 64

**Beats mode — presets**

- Clean breakbeats: Preserve Transients, Transient Loop Mode Off, Transient Envelope 100
- Stuttered rolls: Preserve 1/32, Loop Mode Loop, Envelope 60

**Texture — presets**

- Pad from a drone: Grain 50 ms, Flux 60
- Noise wash from anything: Grain 10 ms, Flux 100

**Re-Pitch**

- Chipmunk soul: +4 to +7 semitones
- Half-speed tape: −12 semitones

**Pitfalls**

- Complex Pro softens kicks — nudge markers pre-transient or layer Beats
- Beats mode mangles tonal content — never use on vocals, pads, held chords
- Texture with Flux > 0 never repeats identically — bounce before you rely on the print
