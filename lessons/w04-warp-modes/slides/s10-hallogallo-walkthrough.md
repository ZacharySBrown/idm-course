# Walkthrough — Hallogallo drums, four modes

Session view. Four clips, same stem (`hallogallo_drums.wav`), four warp modes. Use the live set `live-sets/w04-warp-ab-bed.als` or build it yourself:

1. **Clip 1 — Beats.** Preserve 1/16, Transient Loop Mode = Off, Transient Envelope 100. Baseline. Kick and snare stay intact.
2. **Clip 2 — Complex Pro default.** Formants 100, Envelope 128. Notice the softened kick. This is the pitfall slide in audio form.
3. **Clip 3 — Complex Pro corrected.** Same settings, but nudge the warp marker ~5 ms before each kick. Alternatively, duplicate to a Beats-mode track, HP 80 Hz, sum at −6 dB underneath.
4. **Clip 4 — Texture.** Grain 30 ms, Flux 70%. The motorik pattern dissolves into an evolving percussive cloud.

Watch the waveform zoom — in Live 12 you can see the transient envelope change visually before you hit play. Cite: `bib:ableton_live_12_manual`. Stem: `stemforge:w04-hallogallo-warp-ab`.
